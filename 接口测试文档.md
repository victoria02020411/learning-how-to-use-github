### 2. 用户注册

```
POST http://localhost:8080/api/v1/auth/register
```

新用户注册接口。注意：用户名和邮箱在数据库中必须唯一，重复注册会返回错误。

**请求 Body（raw → JSON）：**
```json
{
  "email": "testuser02@example.com",
  "password": "Test@1234",
  "fullName": "Test User"
}
```

| 编号 | 发送内容 | 预期返回 | 实际结果 |
|------|----------|----------|----------|
| TC-API-02 | 填写合法的用户名、邮箱、密码 | HTTP 200，返回 token 和用户信息 | ✅ 返回 token，注册成功 |
| TC-API-03 | 使用已注册的邮箱再次注册 | HTTP 400，提示邮箱已存在 | ✅ 返回 `VALIDATION_ERROR: Email already registered` |
| TC-API-04 | 邮箱格式不合法（如 abc123） | HTTP 400，提示邮箱格式错误 | ✅ 返回 `email format is invalid` |
| TC-API-05 | 缺少 password 字段 | HTTP 400，提示参数缺失 | ✅ 返回 `password is required` |
| TC-API-06 | 发送空 JSON：`{}` | HTTP 400，所有必填字段报错 | ✅ 返回 password、fullName、email 均 required |

### 3. 用户登录

```
POST http://localhost:8080/api/v1/auth/login
```

登录成功后返回 token，复制 token 值到 Postman 环境变量，后续接口将自动使用。

**请求 Body（raw → JSON）：**
```json
{
  "email": "testuser02@example.com",
  "password": "Test@1234"
}
```

| 编号 | 发送内容 | 预期返回 | 实际结果 |
|------|----------|----------|----------|
| TC-API-07 | 填写正确的邮箱和密码 | HTTP 200，返回 token | ✅ 返回 token，登录成功 |
| TC-API-08 | 密码错误 | HTTP 400，提示密码错误 | ✅ 返回 `AUTH_INVALID_CREDENTIALS: Invalid email or password` |
| TC-API-09 | 用户名不存在 | HTTP 400，提示用户不存在 | ✅ 返回 `Invalid email or password` |
| TC-API-10 | username 字段留空（空字符串） | HTTP 400，提示不能为空 | ✅ 返回 `Invalid email or password` |
| TC-API-11 | 发送空 JSON：`{}` | HTTP 400，参数校验失败 | ✅ 返回 password、email 均 required |

### 4. 获取当前登录用户信息 🔒

```
GET http://localhost:8080/api/v1/users/me
```

需要在 Headers 里带上 token，否则会返回 401 未授权。

**Headers：**

| Key | Value |
|-----|-------|
| Authorization | Bearer {{token}} |

| 编号 | 发送内容 | 预期返回 | 实际结果 |
|------|----------|----------|----------|
| TC-API-12 | Headers 带有效 token | HTTP 200，返回用户信息 | ✅ 返回用户详情 |
| TC-API-13 | 不带 Authorization Header | HTTP 401，提示未授权 | ✅ 返回 `AUTH_FORBIDDEN: Missing Bearer token` |
| TC-API-14 | token 填写错误内容（如 Bearer abc） | HTTP 401，提示 token 无效 | ✅ 返回 `AUTH_INVALID_CREDENTIALS: Invalid access token` |

### 6. 获取价格报价

```
POST http://localhost:8080/api/v1/pricing/quotes
```

根据传入的套餐代码计算实际费用，会考虑折扣等因素。

**请求 Body（raw → JSON）：**
```json
{
  "hireOptionCode": "H1"
}
```

| 编号 | 发送内容 | 预期返回 | 实际结果 |
|------|----------|----------|----------|
| TC-API-17 | `{"hireOptionCode": "H1"}` | 返回价格 3.00 | ✅ 返回 `finalPrice: 3.00, currency: GBP` |
| TC-API-18 | `{"hireOptionCode": "W1"}` | 返回价格 60.00 | ✅ 返回 `finalPrice: 60.00, currency: GBP` |
| TC-API-19 | `{"hireOptionCode": "INVALID"}` | HTTP 404，提示套餐不存在 | ✅ 返回 `RESOURCE_NOT_FOUND: Hire option not found` |
| TC-API-20 | 缺少 hireOptionCode 字段 | HTTP 400，提示必填参数缺失 | ✅ 返回 `hireOptionCode is required` |

**请求 Body（raw → JSON）：**

  "hireOptionCode": "H1",

}
